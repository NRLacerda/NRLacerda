### 📈 [market-microstructure-c](https://github.com/NRLacerda/market-microstructure-c) — Limit Order Book Engine
High-performance LOB reconstruction engine written in C. Replays NASDAQ TotalView-ITCH (via LOBSTER) historical message data to deterministically rebuild full market depth with strict price-time priority. Achieves hundreds of thousands of events per second with predictable memory usage — designed for backtesting, research, and market microstructure education.

`C` `Market Microstructure` `NASDAQ ITCH` `Order Book` `HFT`
